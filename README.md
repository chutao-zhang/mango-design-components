## Components based on React.

### Install

```js
npm install @mangoui/components
```

### Usage

```js
import { Button } from "@mangoui/components";

function Index() {
  return (
    <div>
      <Button>button</Button>
    </div>
  );
}
```

### Components

- `Alert`
- `Badge`
- `Button`
- `Switch`
