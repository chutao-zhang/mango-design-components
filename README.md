## Components based on React.

### Install

```js
npm install @mango-ui/components
```

### Usage

```js
import { Button } from "@mango-ui/components";

function Index() {
  return (
    <div>
      <Button>button</Button>
    </div>
  );
}
```

### Components

- `Alert`
- `Badge`
- `Button`
- `Switch`
