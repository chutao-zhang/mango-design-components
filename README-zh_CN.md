## 基于 React 的组件库。

### 安装

```js
npm install @mango-ui/components
```

### 使用

```js
import { Button } from "@mango-ui/components";

function Index() {
  return (
    <div>
      <Button>按钮</Button>
    </div>
  );
}
```

### 组件

- `Alert`
- `Badge`
- `Button`
- `Switch`
